# Daily Packing App

A simple, mobile-friendly web application for daily packing operations.  
Supports 8 teams (Team A to Team H), QR code scanning, calendar date picker, and live synchronization with Google Sheets.

---

## Features

- **Team-based interface** – Switch between Team A to Team H easily
- **QR Code Scanning** – Scan Customer Name, PI No., or Part Number using your phone camera
- **Calendar Date Picker** – Easy date selection
- **Status Management** – Toggle between "Under Packing" (Orange) and "Packed" (Green)
- **Live Google Sheets Sync** – All data is saved directly to your "Daily Packing" spreadsheet
- **Fully Responsive** – Works great on mobile phones and tablets
- **Hosted on GitHub Pages** – Free and always accessible
